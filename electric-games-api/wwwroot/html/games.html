Games

Game schema

Key Type Description
Id Int The id of the game
Name String The name of the game
Image String Image of the game
Genre String The games genre
Platform String Platforms which the game is available
Developer String The game developer
Release Date String The release date of the game
Price Int The price of the game
Rating String The games rating

Get all Games

You can access the list of games by using the /game endpoint.

Get a single game

You can get a single game by adding the id as a parameter: /game/{id}

Get multiple games

You can get multiple games by adding an array of ids as parameter: /game/[1,2,3] or /game/1,2,3
