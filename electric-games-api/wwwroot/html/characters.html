Characters

Character schema

Key Type Description
Id Int The id of the character
Name String The name of the character
Image String Image of the character
Game String The game where the character is from
Description String Description of the character

Get all characters

You can access the list of characters by using the /character endpoint.

Get a single character

You can get a single character by adding the id as a parameter: /character/{id}

Get multiple characters

You can get multiple characters by adding an array of ids as parameter: /character/[1,2,3] or /character/1,2,3
